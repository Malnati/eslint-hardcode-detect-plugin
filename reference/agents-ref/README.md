# Agentes de referência

Este diretório reúne agentes de governança, compliance e engenharia usados para padronizar instruções de IA em projetos que reutilizam este portfólio.

Estes arquivos são **referência** (portfólio reutilizável), **não** substituem [`AGENTS.md`](../../AGENTS.md) nem os contratos em [`specs/`](../../specs/). Antes de aplicar caminhos como `docs/<repo-externo>/`, `shared/constants.*` ou relatórios em `docs/review/`, leia o mapeamento normativo em **[`specs/agent-reference-agents.md`](../../specs/agent-reference-agents.md)**. A skill do Cursor [`reference-agents-portfolio`](../../.cursor/skills/reference-agents-portfolio/SKILL.md) descreve o procedimento.

## Como usar (projetos genéricos que reutilizam o portfólio)

1. Copie ou adapte os arquivos para a estrutura do seu repositório.
2. Ajuste caminhos (`docs/<repo-externo>/`, `shared/`, etc.) à política local.
3. Para **este** repositório (eslint-plugin-hardcode-detect), siga sempre [`AGENTS.md`](../../AGENTS.md) e o spec de mapeamento acima.
