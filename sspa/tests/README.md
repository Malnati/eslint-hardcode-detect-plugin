# E2E Playwright (pacote `sspa`)

Cópia autocontida dos specs e utilitários alinhados ao subtree `core/sspa/`. Executar a partir da **raiz do monorepo** com `make sspa-test-e2e` (credenciais em `_secrets/` na raiz; variáveis em `core/sspa/tests/.env` ou ambiente).

Ver [docs/e2e-environment-matrix.md](../../docs/e2e-environment-matrix.md).
